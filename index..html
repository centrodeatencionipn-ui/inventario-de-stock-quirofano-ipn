⚙️ Configurar Google Sheets

Este sistema guarda automáticamente el stock en tu Google Sheet. Seguí estos pasos una sola vez:

  1. Abrí Google Sheets y creá una hoja nueva llamada Inventario
  2. Andá a Extensiones → Apps Script y pegá el script de la guía adjunta
  3. Publicá el script como Aplicación web (cualquiera puede acceder)
  4. Copiá la URL que te da Google y pegala acá abajo
→ Usar modo demostración (sin Google Sheets)
💡
Modo local activo — Los datos se guardan automáticamente en este navegador.
Para sincronizar con Google Sheets, descargá el archivo y abrilo directamente en Chrome (no desde Claude). O presioná "☁ Guardar en Sheets" — se abrirá una pestaña nueva para completar el guardado.
